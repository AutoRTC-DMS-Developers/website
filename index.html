Evidence-based Road Safety: an Automatic Road Traffic Crashes Data Management System

Overview

The AutoRTC-DMS (Evidence-based Road Safety: an Automatic Road Traffic Crashes Data Management System) project aims at designing and bringing to the market innovative tools and applications to record, analyze, and share road traffic crashes information in Rwanda. The main deliverables of the AutoRTC-DMS will include: cross-platform, multi-devices mobile applications for road crashes recording and reporting, a repository of road traffic crashes, a web-based dashboard displaying road traffic crashes statistics, and machine learning algorithms for a deeper analysis on road traffic crashes.

Goal

AutoRTC-DMS aims at reducing losses (lives, injuries, and economic losses) caused by road traffic crashes by designing and bringing to the market innovative tools and applications to record, analyze, and share road traffic crashes data in Rwanda.

Objectives

The specific objectives of this project are:

  • Road traffic crashes reporting: This project will design and implement cross-platform, multi-devices mobile applications for road crashes recording and reporting. The applications shall support different user interaction models to allow access to everyone, including people with disabilities.
  • Share, openly, road traffic crashes data: This project will provide a repository of road crashes data. The repository will be composed of SQL and NoSQL databases allowing the storing of road crashes data in its different forms and formats. Data anonymization techniques will be put in place to guarantee open access to the data while preserving privacy.
  • Road crashes dashboard: In addition to road crashes data access interfaces (data access APIs), this project will provide a web-based dashboard displaying road traffic crashes statistics.
  • Road crashes analytics: Having good-quality, openly-accessible, road crash data is not enough. Those data will be fully useful once used to inform road safety decision makers. This project will design a set of machine learning algorithms for a deeper analysis on road traffic crashes.
